# LODESTAR — Brand Assets

Faceted crystal-star mark (the guiding *lodestar* cut like ore / a gem) with a cyan
orbital ring, in the Elite-Dangerous orange-on-black palette.

## Colors
- Orange (primary): #F5731B   Light: #FFA95A   Deep: #973D0F
- Cyan (accent): #3FDDEF   Deep: #17A2C4
- Background: #0C0E14 → #070809

## Files
### /icons  (app icon, with dark rounded container)
- lodestar-icon.svg  — master vector
- lodestar-icon-16/32/48/64/128/256/512/1024.png
- icon.png           — 1024 copy (electron-builder `build/icon.png`)
- lodestar.ico       — Windows multi-res (16–256)
- lodestar.icns      — macOS (included)

### /mark  (bare mark, transparent — favicons, overlays, watermarks)
- lodestar-mark.svg, lodestar-mark-256/512/1024.png

### /logo  (horizontal one-line lockup: mark + wordmark)
- lodestar-logo-horizontal.svg / .png (transparent, orange wordmark)
- lodestar-logo-horizontal@800.png (smaller)
- lodestar-logo-horizontal-dark.svg / .png (on dark panel)
- lodestar-logo-horizontal-tagline-dark.svg / .png (with tagline)

Wordmark: Orbitron, converted to outlines (no font needed to render the SVGs).

contact-sheet.png shows the icon at 16–256px.
